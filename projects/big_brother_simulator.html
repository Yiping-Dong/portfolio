“Who controls the past controls the future. Who controls the present controls the past.” - 1984 by George Orwell.

This project was inspired by the dystopian novel, 1984 by George Orwell.

I am trying to create the experience of a dictator that is trying to hide the facts of global warming. The user can use the knob to control the temperature displayed on the LCD screen in the world in the box. The user can set a temperature to be safe. When the actual temperature detected is below the safe temperature, it will be directly displayed, but when not, the display temperature will be set by the user. The LED lights change colors depending on whether the temperature is safe or not.
